# #defyhatenow image-framer
Getting started

Clone this repo
```sh
git clone https://github.com/defyhatenow-ewer/image-framer.git 
```

Install all req'd dependencies

 ```sh
  npm install
  ```
Start the local server

 ```sh
  npm run dev
  ```
The image framer app will run locally on
```sh
http://localhost:5173/
```
